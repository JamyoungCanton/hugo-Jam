## 🚌 公交可视化系统

一个用于展示城市公交线路与站点信息的 **Web 数据可视化系统**。

### 📌 项目简介

该系统通过地图与数据可视化技术，对城市公交线路、站点以及运行信息进行可视化展示，使用户能够更加直观地查看公交网络结构。

### ⚙️ 主要功能

- 公交线路地图展示
- 公交站点信息展示
- 线路与站点关系可视化
- 数据图表展示
- 线路查询功能

### 🛠 技术栈

- Vue
- JavaScript
- ECharts
- Web 地图 API

## 🌍 旅游小程序

基于 **Vue3 + UniApp** 开发的旅游推荐小程序。

### 📌 项目简介

该小程序为用户提供旅游景点推荐、路线规划以及旅游信息展示等功能。

### ⚙️ 主要功能

- 景点推荐
- 旅游路线展示
- 用户评论系统
- AI 旅游助手

### 🛠 技术栈

- Vue3
- UniApp
- JavaScript
- REST API
