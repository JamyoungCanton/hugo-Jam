🚌 公交可视化系统 链接到标题

一个用于展示城市公交线路与站点信息的 Web 数据可视化系统。

📌 项目简介 链接到标题

该系统通过地图与数据可视化技术，对城市公交线路、站点以及运行信息进行可视化展示，使用户能够更加直观地查看公交网络结构。

⚙️ 主要功能 链接到标题

  • 公交线路地图展示
  • 公交站点信息展示
  • 线路与站点关系可视化
  • 数据图表展示
  • 线路查询功能

🛠 技术栈 链接到标题

  • Vue
  • JavaScript
  • ECharts
  • Web 地图 API
